## Overview

Movie Haven focuses on:
- Clean and intuitive UI inspired by modern streaming platforms
- Optimized frontend performance
- Zero framework dependency
- Scalable and readable code architecture

This project demonstrates strong fundamentals in **frontend engineering, API integration, UI animations, performance optimization, and responsive design**.

---

## Key Features

### Content Discovery
- Trending movies (daily & weekly)
- Dedicated sections for Movies, TV Shows, and Top Rated content
- Infinite scrolling for seamless browsing

### Hero Experience
- Auto-rotating featured content carousel
- Dynamic backdrop images
- Desktop trailer preview on hover
- Quick access to trailers and detailed information via modal

### Search & Navigation
- Instant multi-search (movies & TV shows)
- Section-based navigation with smooth scrolling
- Sidebar navigation for mobile devices

### Performance Optimizations
- Lazy-loaded images using Intersection Observer
- Debounced search and scroll handlers
- Responsive image delivery for different screen sizes

### Accessibility & Productivity
- Keyboard shortcuts for faster navigation
- Clear visual hierarchy
- Fully responsive across mobile, tablet, and desktop
